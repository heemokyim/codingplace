Raw 텍스트 데이터의 정리를 위해서는 자연어 처리(Natural Language Processing),

회차 학습 내용
1회차 텍스트마이닝 개요: 강의 내용 소개 및 텍스트마이닝 프레임워크
- From text to sparse matrix (KoNLPy)
- Logistic Regression을 이용한 classifier (out of vocabulary)
2~3회차 비지도학습 기반의 한국어 자연어 처리 모듈 개발: 사전(dictionary) 또는 말뭉치(corpus)를 이용하지 않는 데이터 기반 한국어 처리 방법 키워드 및 연관어 분석: 데이터를 기반으로 수행되는 키워드 및 연관어 분석
- 단어 추출 및 토크나이저
- 명사 추출
- 품사 판별기
- N-gram
- KoNLPy, Scapy (또는 nltk) 함께 이용하기		 - PMI(Pointwise Mutual Information)
- 분류기를 이용한 키워드 추출
- Proportion을 이용한 키워드 및 연관어 분석
4회차 텍스트 데이터에 분류 모델 적용: 머신러닝의 분류 모델을 텍스트 데이터에 적용하는 방법 소개, 분류기(classifiers) 개요 및 텍스트 데이터에의 적용 Sequential Labeling을 위한 CRF(Conditional Random Field): 띄어쓰기와 같은 Sequence Data의 Classification
- Logistic & Regularization
- Decision Tree, Naive Bayes, Support Vector Machine, Feed-Forward Network		 - CRF를 이용한 띄어쓰기 오류 수정
- Named Entity Recognition
- Spacing
5회차 단어 및 문서의 벡터 표현 학습 및 시각화: 단어 및 문서를 classification/ clustering에 적합하도록 벡터 표현을 바꾸는 임베딩 방법
- Word Embedding: Word2Vec, Doc2Vec, GloVe, FastText
- 시각화: t-SNE, LLE, ISOMAP, MDS, PCA
6회차 토픽 모델링 및 연관어 분석: 임베딩 방법의 확장 Heuristic Spacing: CRF와 같은 머신러닝 알고리즘을 묘사하는 간단한 알고리즘의 구현
- LDA & pyLDAvis
- IMDB Semantic Search		 - 한국어 띄어쓰기
7회차 CNN(Convolutional Neural Network): 이미지에서 성공적인 퍼포먼스를 보인 CNN의 이론과 이를 이용한 텍스트 분류 모델에 대해 소개
- From Logistic to CNN
- Word / Character Level CNN을 이용한 문서 및 문장 분류
8회차 문서 군집화: 비슷한 문서를 하나의 군집으로 추출하는 군집화 모델 개요 및 텍스트 데이터에의 적용 벡터 인덱서: 대량 텍스트의 빠른 유사도 검색을 위한 기술
- K-means, Hierarchical Clustering
- Clustering Visualization
- Carblog		 - Locality Sensitive Hashing
- Inverted Index
9회차 오탈자 교정: 의미적 유사성을 학습하는 Word Representation과 달리 형태적 유사성을 학습하는 방법 Graph Ranking: 의미적 유사성을 학습할 수 있는 Ranking 기반 키워드 및 핵심 문장 추출 방법
- String Distance: Edit Distance, Cosine, Jaccard, etc.
- Inverted Indexer + 오탈자		 - Page Rank, Sim Rank
- Text Rank를 이용한 키워드 추출
- WorkRank
10회차 RNN(Recurrent Neural Network):RNN의 구조와 학습 딥러닝과 자연어 처리의 최신 연구 동향: 챗봇을 포함한 최근의 자연어처리 연구 동향
- RNN의 구조와 학습
- Language Model

데이터 분석 전문가가 가르칩니다

김현중님

현재 서울대학교 산업공학과 데이터마이닝 연구실에서 박사과정을 밟고 있습니다. 한국어 자연어 처리에 관심이 많으며, 딥러닝 모델들을 어떻게 한국어 자연어 처리 작업에 이용할 수 있을지 고민하고 있습니다. 최근에는 PYCON KOREA 2017에서 ‘노가다 없는 텍스트 분석을 위한 한국어 NLP’라는 주제로 발표하기도 했습니다. 분석가가 가능한 많은 시간을 분석에 이용할 수 있기 위해, 노이즈가 많은 현실 텍스트 데이터로부터 최소한의 노력으로 최대한의 의미를 추출하는 방법들을 고민합니다.

머신러닝이 그렇게 생각보다 높은 벽이 있는 건 아니에요. 마찬가지로 텍스트 데이터를 만지는 것도 아주 복잡한 것들이 필요한 게 절대 아닙니다. 생각보다 ‘로직’이 제일 중요해요. 이전에 머신러닝이 어렵다고 생각하셨던 분들도 머신러닝의 원리를 깨닫고, 그에 따른 ‘로직’이 바로 선다면 그렇게 어렵지 않을 거예요. 본 강의를 통해 ‘문제’라고 생각했던 것을 더 효율적으로 고쳐나갈 자신감을 얻어 가실 수 있도록 최선을 다하겠습니다.
