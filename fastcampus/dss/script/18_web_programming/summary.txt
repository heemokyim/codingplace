------------------------------------------------------------

무선대비 유선이 TCP 성능 더 좋은 이유
    => 무선으로 보내면 간섭이 발생해서 데이터가 손실될 수 있음
    => TCP는 reliable하게 보내야되서 다시 받거나 기다려야 함
    => 그래서 다시 받는동안 시간때문에 유선보다 무선이 느림

UDP는 어디서 쓰나
    => 실시간게임은 Nagle TCP or Reliable UDP
    => 먼 나라, 노이즈가 많아서 렉이 많아 패킷이 손실되도 다시 안받음
    => 그래서 렉 심할 때 가끔 스킬이 안나갔던거구나
------------------------------------------------------------

    => TCP로 보내는 패킷 수를 줄임
    => Nagle's Algorithm

쿠키 VS 세션

    1. 쿠키 => 텍스트로 저장된 클라이언트의 정보
    2. 세션 => 서버와 클라이언트가 연결된 상태
    => 클라이언트에 저장되어 일일이 authentication 필요없이 더 빨리 태스크를 처리하도록 돕는다
            => 주로 그에 대한 정보를 객체형태로 서버에 저장함
